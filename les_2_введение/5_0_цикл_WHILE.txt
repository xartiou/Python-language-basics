3. Цикл WHILE - пока условие TRUE: выполнять действия
	while условие:
	    действие 1
	    действие 2
	
4. Пример использования:
 вывод чисел от 0 до 100
	number = 0
	while number <= 100:
	    print(number)
	    number = number + 1
 вывод чисел от 0 до n
	number = 0
	n = int(input('Введите число n: '))
	while number <= n:
	    print(number)
	    number = number + 1
 вывод четных чисел от 0 до n
	number = 0
	n = int(input('Введите число n: '))
	while number <= n:
	    if number % 2 == 0:
	        print(number)
	    number = number + 1
