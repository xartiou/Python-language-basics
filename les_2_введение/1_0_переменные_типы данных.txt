1. Зачем нужны переменные
	- для хранения данных
	- можно использовать несколько раз
	- можно менять значение и тип переменной 
2. Как объявить переменную
	имя_переменной = значение переменной
	name = 'Kesha'
	age = 2
	period = 3.2
3. Как правильно называть переменные
	верно - name, age, person, request, report и похожее
	неверно - a, b, c, ae, sp

4. Как определять тип переменной
как только присваиваем переменной значение автоматически ей присваивается тип
	print(type{значение переменной})
	<class'тип переменной'>
5. Какие бывают типы переменных
	целое число -              int
	число с плавающей точкой - float
	логический тип -           bool
	пустой тип -               None
	строка -                   str
6. Как привести один тип к другому
	- число к строке str(number)
	- строку к числу int(word)
	и аналогично другие
